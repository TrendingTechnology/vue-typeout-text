# Vue-Typeout-Text
> Type out the text :)

```html
 <typeout-text content="Hello world!" speed="2"/>
```

![Demo](demo.gif?raw=true "Demo")

## Installation

Install with NPM

```sh
npm i vue-typeout-text
```

then import and use with Vue

```
Vue.use(vueTypeoutText)
```
